# 分析プロンプト テンプレート集

AIエージェントへの分析依頼をより効果的にするための、良い例・悪い例のテンプレート集です。

## 共通事項
- 分析を頼むときは、ただ分析の内容を指示するだけではなく、その「背景」や「仮説」、「目的」も入れる
- 「目的」だけを与えて「手段」はAIエージェントに委ねる
- 一度に目的に辿りつこうとしない。小さな分析を積み上げていく。
- 仮に仮説があっても、まずはAIとデータ全体について語り合ってもOK(それを文書に残してもらえば、コンテキストになる)
- 資格情報(グラフ、表など)をどんどん出してもらう。人間はグラフを見るのが得意。何が起きているか、何が起きているように見えるかを把握する。
- 自分の直感や仮説をまずAIに話す。その上で、一緒に仮説を検証していく。
- AIは社内用語は知らない。用語集を作るなど、コンテ木臼とを整備する。ノイズが少なく、質の高いコンテキストが重要
- 明確な目的があるなら、分析設計書を利用する。


---

## 例1: 丸投げ

### ❌ 悪い例

```
分析してください
```

**問題点**
- 目的が不明
- 何を知りたいかわからない
- AIが勝手に分析するしかない

### ✅ 良い例

```
売上データを分析してください。

まず全体傾向を確認し、
その後、売上減少や増加が目立つ商品や地域を見つけてください。

さらに、その要因として考えられる仮説も提案してください。
```

---

## 例2: 曖昧な違和感

### ❌ 悪い例

```
最近なんかおかしいです
```

**問題点**
- 「何」がおかしいのか不明

### ✅ 良い例

```
最近、問い合わせ件数が増えている気がします。

本当に増えているのか確認し、
いつ頃から変化したのか、
どの問い合わせ種別が増えているのか調べてください。
```

---

## 例3: 原因を決めつける

### ❌ 悪い例

```
売上低下は広告のせいですよね？
```

**問題点**
- 仮説を事実として扱っている
- 確証バイアスが入りやすい

### ✅ 良い例

```
売上が低下しています。

広告施策の影響も疑っていますが、
決めつけずに主要な要因候補を幅広く調査してください。
```

---

## 例4: 手段だけ指定

### ❌ 悪い例

```
クラスタリングしてください
```

**問題点**
- 目的不明
- 分析手法だけ指定している

### ✅ 良い例

```
顧客を特徴ごとに分類したいです。

どのような顧客グループが存在するのか探索し、
必要であればクラスタリングなど適切な手法を選択してください。
```

---

## 例5: グラフ作成依頼だけ

### ❌ 悪い例

```
グラフを作ってください
```

**問題点**
- グラフを作ることが目的になっている

### ✅ 良い例

```
売上推移を可視化し、
変化が大きかった期間や異常な動きを特定してください。

必要なグラフは適切に選択してください。
```

---

## 例6: 「何か見つけて」

### ❌ 悪い例

```
何か面白いことを見つけてください
```

**問題点**
- 範囲が広すぎる

### ✅ 良い例

```
事前仮説はありません。

顧客行動や売上の特徴について、
私が見落としている可能性のある興味深いパターンを探索してください。

特に事業上のアクションにつながりそうな発見を優先してください。
```

---

## 例7: 結果だけ要求

### ❌ 悪い例

```
原因を教えてください
```

**問題点**
- 分析プロセスが不透明

### ✅ 良い例

```
売上低下の原因を調査してください。

原因候補を列挙し、
それぞれについて根拠となるデータや分析結果も示してください。
```

---

## 例8: 「比較して」

### ❌ 悪い例

```
比較してください
```

**問題点**
- 比較対象が不明

### ✅ 良い例

```
2025年と2026年を比較し、
売上・顧客数・客単価の違いを分析してください。

特に変化の大きい領域を特定してください。
```

---

## 例9: ドメイン知識を伝えない

### ❌ 悪い例

```
解約率を分析してください
```

**問題点**
- 現場が知っている背景情報が欠落している

### ✅ 良い例

```
解約率を分析してください。

現場では、
競合製品の登場が影響しているのではないかと言われています。

ただし、その仮説に限定せず幅広く要因を探索してください。
```

---

## 例10: 深掘りを止めてしまう

### ❌ 悪い例

```
結果を教えてください
```

**問題点**
- 一回で終了してしまう

### ✅ 良い例

```
分析結果を示してください。

また、
その結果から次に確認すべき観点や、
考えられる仮説も提案してください。
```
